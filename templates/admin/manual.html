How to Use AIA - CAST

  1. Connect the imaging devices to the Raspberry Pi using HDMI cables.
  2. Connect the Raspberry Pi to the local network via WiFi connection or using a network cable. Check the local IP address of the Raspberry Pi as it will be used to access the AIA - CAST. If necessary, manually configure the IP address. Note: For use in the SMS232 room, the IP address currently configured is: 10.0.128.129
  3. Connect the Administrator and/or User's computer to the same network as the Raspberry Pi.
  4. To create an access session for the application, access the address http://RASPBERRY_PI_IP:5000 (If you have not changed the IP address, the address is: http://10.0.128.129:5000).
  5. Log in using the Administrator account.
  6. After logging in, the Administrator has two options:
    • Start the CAST from your own computer.
    • For the user, create a ticket and select the maximum time for which the CAST session will be active and inform the User of the Ticket number generated.
  7. Using a browser, the user must access the Raspberry Pi's IP address: http://RASPBERRY_PI_IP:5000 (If you have not changed the IP address, the address is: http://10.0.128.129:5000).
  8. On the login screen, the user must use the Ticket provided to access.
  9. After logging in, the user must click on the button to start the CAST.
  10. When clicking on the CAST button, an .EXE file will be downloaded.
    • Note: In some browsers it is common for the .EXE file to be blocked after downloading for security reasons. If this occurs, follow the browser's instructions to execute even with the security alert.
  11. When you finish downloading the .EXE file, run this file.
  12. When executing the .EXE file, it will automatically check whether the user's computer has the necessary files to use AIA - CAST. If the computer has the necessary files, the CAST service will start automatically.
  13. If the user's computer does not have the necessary files, they will be automatically installed via the .EXE file.
  14. After installing the files, run the .EXE file again to initialize the CAST service.
  15. The session will end automatically when the time selected by the Administrator reaches the end or when the user closes the application window.
